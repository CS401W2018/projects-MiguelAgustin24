Questions

1. Explain the difference between inline, embedded, and external CSS. - Inline CSS: Styles are applied directly to specific elements using the style attribute.
- Embedded CSS: Styles are placed inside the style tag within the HTML document.
- External CSS: Styles are stored in a separate CSS file that is linked to the HTML document.

2.Explain why you should only use external CSS.
- External CSS helps maintain a clear separation between content (HTML) and design (CSS). This makes your code more organized and easier to manage.

3.What is a selector used for?
- It is used to target the HTML elements on our web pages that we want to style.
