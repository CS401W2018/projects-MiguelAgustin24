Engagement Questions

  1. What you learned about Flex and CSS.
    • CSS is the broader styling language used for styling websites. Flexbox is a specific layout model within CSS that helps create flexible and responsive layouts more easily.
  2. Why do you need to have an accessible website? What does that mean?
    • An accessible website is one that is designed to be used by all people, including those with disabilities. This means that the website is designed in a way that makes it easy for people with disabilities to navigate and interact with the content.
