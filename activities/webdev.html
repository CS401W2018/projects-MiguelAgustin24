Questions:
  1. Why is the viewport meta tag important?
    • It helps the browser to scale the page to fit the screen
  2. How can you adjust the CSS for printing?
    • You can create a separate CSS file for printing, and then link it to the HTML file using the media attribute.
  3. When would you want to adjust the CSS for printing?
    • You would want to adjust the CSS for printing when you want the printed version of the page to look different from the screen version.
