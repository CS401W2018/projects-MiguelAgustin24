QUESTIONS

  1. What is the difference between a block and an inline tag? Give examples of each one. Answer: Block-level elements take up the full width available and start on a new line. Examples include:
    • <div>
    • <p>
    • <h1> to <h6>
    Inline elements only take up as much width as necessary and do not start on a new line. Examples include:
    • <span>
    • <a>
    • <img>
  2. What is normal flow in HTML? Answer: Normal flow is the default layout of the page. Elements are displayed in the order they appear in the HTML document.
  3. What does float do? Why do you need to clear the float? Answer: Float moves an element to the left or right of its container. You need to clear the float to prevent other elements from wrapping around the floated element.
  4. What do absolute, fixed, and relative do? Answer:
    • Absolute positions relative to the nearest positioned ancestor or the page itself, out of the normal flow.
    • Fixed positions an element relative to the viewport.
    • Relative positions an element relative to its normal position.
  5. How can you make nice navigation buttons for mobile devices? Answer: You can use CSS to style the navigation buttons with padding, margins, and colors. You can also use hover effects to make the buttons interactive.
  6. Where can you find videos to review what we talked about in class? Answer: You can find videos on YouTube, or Canvas.
