  1. Why don't we use the border property?
    • We don't use the border property because it is not necessary to use it. The border property is used to create a border around an element. In this case, we don't need to create a border around the table because we can't edit it inside the css file..
  2. What is the easiest way to color every other row a different color?
    • The easiest way to color every other row a different color is by using the nth-child() selector. We can use the nth-child() selector to select every other row and apply a different background color to it.
  3. When should we use a table?
    • We should use a table when we need to display tabular data. Tables are used to organize and display data in rows and columns. They are useful for presenting data in a structured and organized way.
