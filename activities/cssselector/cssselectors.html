Questions

1. List out the different ways to select an item.
We can use #id, class, div, and a span.
2. What is the difference between an ID and a Class?
An ID can only be used once, while a class can be used many times. That's why web designers prefer to use class instead of an ID.
3. What is the difference between a SPAN tag and a DIV tag?
A div element is used for block-level organization and styling of page elements, whereas a span element is used for inline organization and styling.
4. What is the difference between a DIV tag and HEADER, MAIN, FOOTER?
The DIV tag is a generic container for grouping content, while HEADER, MAIN, and FOOTER are piece of code specifically used to define the header, main content, and footer sections of a webpage, respectively.
