Internet

It is a global network of interconnected computers that allows computers to
communicate with each other, allowing people to share and access information all across the globe.

Web

The Web is the websites that the people use when they access the internet.
It is commonly seen in Web Browsers such as Google Chrome or Mozilla Firefox to show information so that people can read the display.

How are the Internet and Web different?

The internet is the foundation of the communication in each of the computer. We cannot access a website, social media or read through emails without accessing the internet.
On the other hand, a web is the result of using the internet, people can gain information from different sites.

ASCII and Unicode

ASCII - it is a way to represent letters, numbers, and symbols using the language that the computer understands.
Unicode -it is a way to represent character from all languages and symbols on a computer. Unicode can access almost all characters from different programs and devices.

What is UTF-8?

It is an encoding standard to represent any Unicode character.
It can handle many different languages.

URL

It is an acronym meaning, Uniform Resource Locator. Using a URL will tell the browser where to find a specific page or file on the internet.

Domain Name

A Domain name is the name that we used to a website. It is a more user-friendly way to remember the IP address of a website.

HTTP

Hypertext Transfer Protocol - it tells web browsers and servers how to communicate and share web pages and data.

HTTPS

Hypertext Transfer Protocol Secure - a more secure version of HTTP that encrypts the data sent by the HTTP. It keeps information safe from hackers.

IP Address

Internet Protocol - a unique number separated either by periods for IPv4 and colons for IPv6 to indentify a device connected on a network.
It allows devices to communicate inside the internet.

DNS

Domain Name System - it translates domain names into IP address so that browsers can gain access to websites and other internet resources.

W3C Recommendations

These are guidelins created by the World Wide Web community to create a standard so that users can have smoother access to sites and to ensure consistency on the internet.

XML, XHTML, HTML, HTML5

XML (Extensible Markup Language) - it is used to store and transport data in such a way that both people and computers can read.


XHTML (Extensible Hypertext Markup Language) - it is a stricter version of HTML while following the rules of XML. This makes sure that web pages are consistent and easier to read for computers.


HTML (Hypertext Markup Language) - A language used to create webpages and websites. It uses tags to add text, images, links, and other elements to a page.


HTML5 - it is the latest version of HTML with added featurs for creating modern websites.
